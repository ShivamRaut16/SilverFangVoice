# Built With SentinelVoice AI

## Tech Stack

### Languages & Frameworks
- **TypeScript**: Ensuring type safety across the neural pipeline.
- **React 19**: Modern UI component architecture.
- **Vite**: Ultra-fast build tool and development server.
- **Vanilla CSS**: Custom-crafted design system with glassmorphism and animations.

### Google Cloud Products
- **Gemini 2.5 Flash**: Powering the high-speed "Live Neural Bridge" for real-time audio/semantic processing.
- **Gemini 3 Pro Preview**: Driving the "Intent Snapshot" mode for deep reasoning and multi-pass semantic verification.
- **Google Generative AI SDK (@google/genai)**: The core interface for model interaction.

### Other Tools & APIs
- **ElevenLabs API**: High-fidelity, expressive text-to-speech engine.
- **Vercel**: Optimized hosting and deployment infrastructure.
- **FontAwesome**: Premium iconography for the neural interface.
- **Web Audio API**: Low-latency audio processing and visualization.
